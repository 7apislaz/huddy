huddy 캐릭터를 선택합니다. 13종 캐릭터 중 원하는 버디를 고를 수 있습니다.

전체 캐릭터를 미리보려면:
```bash
huddy select
```

특정 캐릭터를 바로 선택하려면:
```bash
huddy select $ARGUMENTS
```

사용 가능한 캐릭터: peekcat, duck, cat, blob, penguin, rabbit, dog, hamster, owl, octopus, ghost, dragon, axolotl

예시: `/huddy-select peekcat` — 브랜드 마스코트 고양이 (기본값)
